# 🚀 Next-Gen Full-Stack 3D Portfolio

A professionally structured, reusable portfolio template with a separate **Next.js Frontend** and **Express.js Backend**.

## 🏗️ Project Structure
- **/frontend:** Next.js 14+ (App Router). Fully responsive, 3D animated landing page.
- **/backend:** Express.js modular API. Handles contact forms and business logic.

## 🏃 Quick Start

### 1. Backend (API)
```bash
cd backend
npm install
npm start
```

### 2. Frontend (UI)
```bash
cd frontend
npm install
npm run dev
```

## 🛠️ Configuration
All project details (Name, Bio, Links, Images) are managed in one file:
`frontend/src/data/config.ts`

## 💎 Features
- **Fluid Responsiveness:** Perfectly balanced layouts for all device sizes.
- **3D Hero Scene:** Smooth `@react-three/fiber` starfield background.
- **API Integrated Contact Form:** Frontend connects seamlessly to the local backend.
- **Premium Aesthetics:** Neon-cyberpunk theme with glassmorphic UI.
